Hold shift and hit i to open up the cheat code line.


fullscreen - makes it fullscreen (might make it windowed if already fullscreen).
720 - changes resolution to 720p.
1080 - changes resolution to 1080p.
exittutorial - ends the tutorial.
deletesave - deletes save, and restarts the game.
tearstowine - adds 10000 gold.
spawnworm - spawns the boss worm.
showcustomerimage - shows the customers image for the selling screen.
